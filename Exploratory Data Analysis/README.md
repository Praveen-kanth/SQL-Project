# Exploratory Data Analysis (EDA) using SQL

Welcome to the Exploratory Data Analysis (EDA) repository! This project focuses on analyzing and understanding a dataset using SQL queries to uncover patterns, trends, and valuable insights.

## 📌 Project Overview

In this project, I performed exploratory data analysis to gain meaningful insights from the dataset by:

* Examining data distributions
* Identifying trends and patterns
* Analyzing key performance metrics
* Detecting outliers and anomalies
* Performing aggregations and statistical summaries
* Answering business-related questions using SQL

The objective was to transform raw data into actionable insights for data-driven decision-making.

## 🛠️ Tools & Technologies

* CSV Dataset
